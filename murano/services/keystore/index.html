Key-Value Store

A Key Value storage Service.
Moreover, this service provides most of the powerful features of a Redis database.
Note: the number of keys you can use depend on your Murano plan. Use list operation to see your current quota and usage.

Operations

  • Keystore.info() - Get namespace information
  • Keystore.list() - Get namespace keys
  • Keystore.clear() - Clear solution
  • Keystore.get() - Get a value
  • Keystore.set() - Set a value
  • Keystore.delete() - Delete key
  • Keystore.command() - Execute a Redis command

Operations

info

Returns namespace information including usage and quota

Responses
Solution namespace information - object
Name Type Description
quota object Usage limitation applying to this service
quota.keys integer Maximum number of keys allowed for this solution depending on your Murano plan. Contact Murano team for more information.
usage object Namespace usage
usage.keys integer Current number of keys being used by the solution namespace.
Error response - object
Name Type Description
code string Error code
error object, string Error details
message string Error Message
Example

response = Keystore.info()

list

Returns namespace keys

Responses
Solution namespace information - object
Name Type Description
keys [ string ] Active namespace keys
quota object Usage limitation applying to this service. WARNING! This property is deprecated and will be removed in a future version. Use the info operation instead to get this value.
quota.keys integer Maximum number of keys allowed for this solution depending on your Murano plan. Contact Murano team for more information.
usage object Namespace usage. WARNING! This property is deprecated and will be removed in a future version. Use the info operation instead to get this value.
usage.keys integer Current number of keys being used by the solution namespace.
Error response - object
Name Type Description
code string Error code
error object, string Error details
message string Error Message
Example

response = Keystore.list()

clear

Remove all keys from this solution. Be care, there is no going back!

Responses
All solution keys removed - nil
Error response - object
Name Type Description
code string Error code
error object, string Error details
message string Error Message
Example

response = Keystore.clear()

get

Returns the value of the given key

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
key ^[\w\@.-]+$ Key id
Responses
The response object containing the key value. - object
Name Type Description
value array, object, string, number, boolean, null Value data
Error response - object
Name Type Description
code string Error code
error object, string Error details
message string Error Message
Example
local parameters = { 
 key = "Y0w7qOZpxhWG.9mfZlGTIE-xRCqkDVJiaJgebA9qOs5aAUj4O"
 }

response = Keystore.get(parameters)

set

Set or update a key value. If the key doesn't exist, this operation will increase your usage count.

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
key ^[\w\@.-]+$ Key id
value string, number, boolean, null Value data
Responses
Key successfully set - nil
Namespace, Key count quota reached. Delete some un-used keys or contact Murano team for upgrading your plan. You can view your current namespace usage with the list operation or on your Murano portal account.
  • nil
Error response - object
Name Type Description
code string Error code
error object, string Error details
message string Error Message
Example
local parameters = { 
 key = "2BDSsV8XtrYdr58t.DJ72m9ljz03myUnZn1ttlEX9126tCNS1UY5cAAAa", 
 value = -32027822
 }

response = Keystore.set(parameters)

Name Type Description

Key successfully deleted - nil
Error response - object
Name Type Description
code string Error code
error object, string Error details
message string Error Message
Example
local parameters = { 
 key = "oFYsZNB"
 }

response = Keystore.delete(parameters)

command

This function offers some popular Redis commands to execute on the key. Please refere to http://redis.io/commands documentation page for the command information and parameters. Remember, some command will increase your namespace usage and may hit your namespace quota. Not all Redis command are supported please refere to the command parameter information.

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
key ^[\w\@.-]+$ Target key
command "get", "set", "setnx", "strlen", "type", "exists", "append", "incrby", "decrby", "getset", "incrbyfloat", "lpush", "lindex", "llen", "linsert", "lrange", "lrem", "lset", "ltrim", "rpop", "rpush", "sadd", "scard", "smembers", "spop", "setex", "psetex", "expire", "pexpireat", "pexpire", "expireat", "pttl", "ttl", "persist", "geoadd", "geohash", "geodist", "geopos", "georadius" This function offers some popular Redis commands to execute on the key. Please refere to http://redis.io/commands documentation page for the command information and parameters.
args [ string, number, boolean ] List of arguments for the specified command. Please refere to http://redis.io/commands documentation page for the command information and parameters.
Responses
The response object containing the key value. - object
Name Type Description
value array, object, string, number, boolean, null Value data
Namespace, Key count quota reached. Delete some un-used keys or contact Murano team for upgrading your plan. You can view your current namespace usage with the list operation or on your Murano portal account.
  • nil
Error response - object
Name Type Description
code string Error code
error object, string Error details
message string Error Message
Example
-- Add a string to a list
result = Keystore.command({
  key = "myList",
  command = "lpush",
  args = {"oneItem"}
})
-- Retrieve the list content
result = Keystore.command({
  key = "myList",
  command = "lrange",
  args = {0, -1}
})

Time Series Storage Service

A time series storage using InfluxDB

Operations

  • Timeseries.query() - query the database
  • Timeseries.write() - writes data into database
  • Timeseries.command() - executes a command against the database

Operations

query

query the database

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
q string Query string (https://docs.influxdata.com/influxdb/v0.13/query_language/)
epoch string Timestamps in Unix epoch format
chunk_size string Explicitly set the batch size
Responses
Database information retrieved - nil
Error - object
Name Type Description
message string Error Message
status_code string Error code
Example
local parameters = { 
 q = "aute in anim enim in", 
 ["chunk_size"] = "aute"
 }

response = Timeseries.query(parameters)

write

writes data into database

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
query string Write data query (https://docs.influxdata.com/influxdb/v0.13/write_protocols/write_syntax/)
Responses
Data successfully inserted - nil
Error - object
Name Type Description
message string Error Message
status_code string Error code
Example
local parameters = { 
 query = "ut eu"
 }

response = Timeseries.write(parameters)

command

executes a command against the database

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
q string Query string (https://docs.influxdata.com/influxdb/v0.13/query_language/)
epoch string Timestamps in Unix epoch format
chunk_size string Explicitly set the batch size
Responses
Database information retrieved - nil
Error - object
Name Type Description
message string Error Message
status_code string Error code
Example
local parameters = { 
 q = "Ut Excepteur dolore nulla", 
 ["chunk_size"] = "ullamco velit laborum", 
 epoch = "irure enim ipsum"
 }

response = Timeseries.command(parameters)
